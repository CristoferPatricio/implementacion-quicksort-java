Prompt utilizado para generar el algoritmo QuickSort

A continuación se muestra el prompt o instrucción exacta que se le puede dar a una inteligencia artificial (como ChatGPT o Gemini) para que genere el código fuente proporcionado en este repositorio:

Usuario:

"Actúa como un experto en estructuras de datos y algoritmos en Java. Por favor, escribe la implementación del algoritmo QuickSort en Java. Incluye lo siguiente:

Un método de ordenamiento bien documentado usando la técnica de partición (con el último elemento como pivote).

Una clase estática interna llamada Stats para contar y retornar el número de particiones (llamadas recursivas), comparaciones e intercambios.

Un método main que pruebe el algoritmo con arreglos de números aleatorios de tamaños desde 10 hasta 200 (en incrementos de 10).

Haz que el método main imprima las estadísticas en formato CSV (tamaño, particiones, comparaciones, intercambios) para que pueda llevar esos datos a Excel y hacer una gráfica."
